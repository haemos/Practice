  ### 算法
  BF（Brute Force）算法，是一种在字符串匹配的算法中，比较符合人类自然思维方式的方法，即对源字符串和目标字符串逐个字符地进行比较，直到在源字符串中找到完全与目标字符串匹配的子字符串，或者遍历到最后发现找不到能匹配的子字符串。算法思路很简单，但也很暴力。
  
 ### code
 ~~~ c++
 int BF(String S, String T, int p){
    //记录主串s中当前位置的下标
    int i = p;
    //用于记录子串的下标
    int j = 1;
    //若i小于S的长度并且j小于T的长度时，循环继续
    while (i<=S[0] && j<=T[0]) {
        //比较两个值是否相等
        if(S[i] == T[j]){
            i++;
            j++;
        }else{
            //回退指针
            i = i-j+2;
            j = 1;
        }
    }
    //如果j>T[0],匹配成功
    if(j>T[0]){
        return i-T[0];
    }else{
        return -1;
    }
}
~~~
